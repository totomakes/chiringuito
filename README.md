# 🏖️ Chiringuito Menu

A premium, modern, and interactive digital menu for your beach bar or restaurant. Built with high-performance technologies to provide a seamless experience for your customers.

![Chiringuito Logo](./ChirinLogo.svg)

## ✨ Features

- **🔍 Smart Search**: Instantly find drinks by name, ingredients, or tags.
- **🏷️ Category Filtering**: Easily navigate through different drink sections.
- **🔥 Featured Items**: Highlight your best-selling or seasonal drinks at the top.
- **🎲 "¡Sorpréndeme!" (Surprise Me)**: A fun way for customers to discover new drinks with a single click.
- **📱 Fully Responsive**: Optimized for mobile, tablet, and desktop views.
- **🎨 Premium Design**: Dark mode aesthetic with neon accents and smooth animations.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/robbur/Chiringuito_menu.git
   cd Chiringuito_menu
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/App.tsx`: The main application component with all logic and layout.
- `src/components/ChirinLogo.tsx`: The animated SVG logo component.
- `menu.json`: The central data file for all menu items and categories.
- `src/index.css`: Global styles and Tailwind configuration.
